# Cafetería Triple Espresso

La finalidad de este proyecto fue la de crear un sitio web para la cafetería ficticia: Triple Espresso. El objetivo fue presentar información clave tales como 2 recetas, un formulario para realizar una reservación y al final una sección de contacto, todo esto siguiendo el brief, etapa por etapa, del 4to sprint en TripleTen.

# Tecnologías usadas

Este proyecto fue desarrollado utlizando:

- HTML
- CSS

# Descripción del proyecto

Este sitio web contiene 4 secciones en total:

- Encabezado: Aquí tenemos el título y descripción de nuestra cafetería, un menú de navegación funcional, una imagen que plasma el interior de una cafetería, así como también los horarios de la cafetería y su dirección.

- Sección "Recetas": En este apartado podemos encontrar dos videos embebidos de recetas en YouTube.

- Sección "Formulario de Reservación": Es en esta sección del sitio web donde podemos realizar una reservación mediante el llenado de un formulario.

- Pie de página: Finalmente, tenemos nuestro pie de página con el logo de la cafetería, los enlaces a las redes sociales y el nombre del autor.

# Planes de mejora

Se planea a futuro agregar más secciones y funcionalidades al sitio web, tales como:

- Sección de menú de la cafetería
- Sección "Nuestros clientes dicen esto de nosotros..."
- Funciones para permitir la compra en línea si se reside en las cercanías de la cafetería.
- Scripts que agreguen una mayor funcionalidad y dinamismo al sitio web.
